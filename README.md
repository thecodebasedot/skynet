# SkyNet Viewer

SkyNet Viewer is a high-performance, next-generation remote display system designed for speed and efficiency. It allows you to view and interact with desktop environments running on remote computers with unparalleled clarity and responsiveness.

## Features

- **High Speed:** Optimized for image-intensive applications.
- **Secure:** Advanced security protocols.
- **Cross-Platform:** Works seamlessly across major operating systems.
- **User-Friendly:** Intuitive interface designed for professionals.

## Installation

To run SkyNet Viewer, simply execute the `VncViewer.jar` file.

```bash
java -jar VncViewer.jar
```
